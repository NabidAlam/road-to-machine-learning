# Full-stack track — examples scratch space

This folder is for **your own experiments** while following:

- [Full-stack track lessons](../../resources/full_stack_track/README.md)
- [Full-Stack AI Engineer Blueprint](../../resources/full_stack_ai_engineer_roadmap.md)

## Conventions

- Do **not** commit secrets. Use `.env` locally and add local-only paths to `.gitignore` if needed.
- Prefer small subfolders per experiment, for example `01-hello-express/` with its own `README.md` describing how to run it.
- Keep dependency choices explicit (`package.json` per experiment is fine).

## Relationship to the curriculum

The numbered **`00`–`25`** module folders remain the canonical ML curriculum. This directory is optional and learner-owned.
