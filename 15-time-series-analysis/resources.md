# Time Series Analysis — Additional Resources

Curated resources to deepen your time series skills.

## Books

- **Forecasting: Principles and Practice** (Hyndman & Athanasopoulos) — the most practical reference

## Courses / Lectures

- StatQuest (time series + ARIMA intuition)
- Andrew Ng / Deep Learning sequences material (for RNN/LSTM intuition)

## Tooling docs

- `statsmodels` time series docs (ARIMA, seasonal decomposition)
- `prophet` docs (trend/seasonality components)

## Practice datasets (public)

- Airline passengers (classic)
- Electricity / energy demand datasets
- Retail sales datasets
