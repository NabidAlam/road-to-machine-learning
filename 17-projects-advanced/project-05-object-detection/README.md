# Project 5: Object Detection

Detect and localize multiple objects in images.

## Difficulty
Advanced

## Time Estimate
2-3 weeks

## Skills You'll Practice
- Computer Vision
- Object Detection
- YOLO
- R-CNN

## Learning Objectives

- Understand object detection architectures
- Implement YOLO
- Use R-CNN family
- Handle bounding box regression
- Evaluate with mAP

## Dataset

[COCO Dataset](https://cocodataset.org/) or custom dataset

## Project Steps

1. Prepare object detection dataset
2. Implement YOLO
3. Try R-CNN variants
4. Train detection model
5. Evaluate with mAP
6. Real-time detection
