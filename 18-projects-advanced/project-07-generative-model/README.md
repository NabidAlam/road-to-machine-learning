# Project 7: Generative Model (GAN/VAE)

Build a generative model to create new images or text.

## Difficulty
Advanced

## Time Estimate
2-3 weeks

## Skills You'll Practice
- Generative Models
- GANs
- VAEs
- Deep Learning

## Learning Objectives

- Understand GANs and VAEs
- Train generative models
- Handle training challenges
- Evaluate generative models
- Generate new content

## Dataset

CelebA, MNIST, or custom dataset

## Project Steps

1. Understand GAN/VAE architecture
2. Build generator and discriminator
3. Train model
4. Generate new samples
5. Evaluate quality
6. Experiment with variations

- [VAE Tutorial](https://www.tensorflow.org/tutorials/generative/cvae)
